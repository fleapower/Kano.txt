Keyboard Shortcuts & Help

Navigation
↑/↓ Navigate tasks
Shift + ↑/↓ Select multiple tasks
Ctrl + ↑/↓ Move task up/down
Task Management
Space Edit selected task
( Add new task
d Delete selected task(s)
x Toggle task completion
Filters & Organization
c Cycle completion state filter
u Cycle due date filter
q Cycle priority filter
0 Clear all filters
Priority & Dates
Alt + ↑/↓ Increase/decrease priority
i Set priority for selected task(s)
s Set due date for selected task(s)
p Postpone selected task(s)
Filter Sets
1-9 Load filter set
Ctrl + Alt + 1-9 Save current filters as set
Other
f Focus search box
Ctrl + a Select all tasks
Ctrl + c Copy selected tasks
r Sort by priority
t Sort by due date
h Show this help window
